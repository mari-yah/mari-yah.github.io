Project

Radio Watch

An e-commerce site with a smart Bargain Feature, product filtering, and PDF bill generation. Built using Django.
GitHub Repo


Radio Watch is an e-commerce website with a modern interface and a unique Bargain Feature that offers personalized discounts based on user purchase history.

The website includes features such as advanced product filtering, intuitive cart management, seamless checkout, and automatic PDF bill generation. This project showcases a full-stack implementation with real-world business logic.

Technologies Used

Django Python HTML5 CSS3 JavaScript MySQL
