Sowinz.com

A live e-commerce store for a premium organic skincare and haircare brand.
Live: Visit Site

Sowinz.com is a Shopify-based e-commerce website built for a premium skincare and haircare brand that focuses on organic, chemical-free products.

The website includes custom-designed sections, product bundles (e.g., 2-combo options), discount code features, UPI/COD payment integration, and SEO-optimized content. I also handled visual content using Canva and Photoshop, set up affiliate marketing with GoAffPro, and connected Meta Business tools for ad analytics and shop integration.

Technologies & Tools Used

Shopify Liquid HTML CSS Canva Photoshop GoAffPro Meta Business Suite
